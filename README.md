# TaskNote

## Setup Instructions

1.  **Clone the repository:**

    `git clone <repository-url>  `
    
    `cd <project-folder>` 
    
2.  **Install dependencies:**

    `npm install` 
    
3.  **Start the development server**

    `npm run dev` 
    
4.  **Build it for production (if you want):**

    `npm run build` 
    
5.  **Preview the production build:**

    `npm run preview`
